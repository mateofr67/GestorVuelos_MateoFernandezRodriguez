Pantalla Principal

La pantalla principal de la aplicación proporciona acceso rápido a las siguientes secciones:

Gestor Compañías

El Gestor de Compañías le permite realizar operaciones como dar de alta, dar de baja, consultar y modificar información relacionada con las compañías aéreas.

Gestor Vuelos Base

El Gestor de Vuelos Base le permite realizar operaciones similares para gestionar la información de los vuelos base.

Gestor Vuelos Diarios

El Gestor de Vuelos Diarios le brinda funcionalidades para manejar información específica de los vuelos diarios, como altas, bajas, consultas y modificaciones.

Detalles Operativos

En la sección de Detalles Operativos, puede acceder a información detallada, como la visualización de aeropuertos, salidas, llegadas, recaudaciones y vuelos por compañía.
