Baja de Vuelos Base

En la sección de "Baja de Vuelos Base", sigue estos pasos para eliminar un vuelo base:

Seleccionar Vuelo Base a Dar de Baja:

Utiliza el desplegable para seleccionar el código del vuelo que deseas dar de baja.

Dar de Baja:

Después de seleccionar el código de vuelo , haz clic en el botón correspondiente para eliminarlo.
