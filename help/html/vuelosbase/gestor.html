Gestor de Vuelos Base

El Gestor de Vuelos Base le permite realizar diversas operaciones relacionadas con los vuelos base:

Alta de Vuelos Base

La opción "Alta" le permite agregar un nuevo vuelo base.

Baja de Vuelos Base

La opción "Baja" le permite eliminar un vuelo base existente.

Consulta de Vuelos Base

La opción "Consulta" le permite ver información detallada sobre los vuelos base registrados.

Modificación de Vuelos Base

La opción "Modificación" le permite actualizar la información de un vuelo base existente.
