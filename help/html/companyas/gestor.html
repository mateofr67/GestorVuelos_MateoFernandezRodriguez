Gestor de Compañías

El Gestor de Compañías le permite realizar diversas operaciones relacionadas con las compañías aéreas:

Alta de Compañías

La opción "Alta" le permite agregar una nueva compañía a la base de datos.

Baja de Compañías

La opción "Baja" le permite eliminar una compañía existente de la base de datos.

Consulta de Compañías

La opción "Consulta" le permite buscar y ver información detallada sobre una compañía específica.

Modificación de Compañías

La opción "Modificación" le permite actualizar la información de una compañía existente en la base de datos.
